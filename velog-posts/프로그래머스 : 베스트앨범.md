<h3 id="1트-오답">1트 (오답)</h3>
<p>→ 장르별 총 스트리밍 횟수를 <code>genMap</code>에 저장하고, <code>TreeMap</code>의 <strong>Key 기준 역순 정렬</strong>로 장르 순서를 정하려 했다.<br />→ 각 곡의 인덱스는 <code>playIdx</code>에 <code>plays[i] → i</code> 형태로 저장했다.</p>
<pre><code class="language-java">import java.util.*;
// 음악 클래스
class Music implements Comparable&lt;Music&gt; {
    String genre;
    int play;

    public Music(String genre, int play) {
        this.genre = genre;
        this.play = play;
    }

    @Override
    public int compareTo(Music m) {
        return m.play - this.play;
    }
}
class Solution {
    public int[] solution(String[] genres, int[] plays) {
        Map&lt;String, Integer&gt; genMap = new HashMap&lt;&gt;(); // 장르별 스트리밍 횟수 저장
        Map&lt;Integer, Integer&gt; playIdx = new HashMap&lt;&gt;(); // 스트리밍 횟수 인덱스 저장
        List&lt;Music&gt; playlist = new ArrayList&lt;&gt;(); // 곡별 스트리밍 횟수 저장

        for(int i=0; i&lt;plays.length; i++) {
            genMap.put(genres[i], genMap.getOrDefault(genres[i],0) + plays[i]);
            playIdx.put(plays[i], i);
            playlist.add(new Music(genres[i], plays[i]));
        }

        // 새로 알게 된 개념 -&gt; Key 기준으로 정렬
        Map&lt;String, Integer&gt; genSort = new TreeMap&lt;&gt;(Collections.reverseOrder());
        genSort.putAll(genMap);

        // 리스트에 장르별 베스트 2개 담기
        List&lt;Integer&gt; list = new ArrayList&lt;&gt;();

        for(Map.Entry&lt;String, Integer&gt; gen : genSort.entrySet()) {
            List&lt;Music&gt; tmp = new ArrayList&lt;&gt;();

            for(Music m : playlist) {
                if(gen.getKey().equals(m.genre)) tmp.add(m);
            }

            Collections.sort(tmp);

            if(tmp.size() == 1) {
                list.add(playIdx.get(tmp.get(0).play));
            }
            else {
                list.add(playIdx.get(tmp.get(0).play));
                list.add(playIdx.get(tmp.get(1).play));
            }
        }

        int[] answer = new int[list.size()];
        for(int i=0; i&lt;list.size(); i++) answer[i] = list.get(i);

        return answer;
    }
}</code></pre>
<h4 id="위-풀이의-결과">위 풀이의 결과</h4>
<ul>
<li><code>TreeMap</code>의 역순 정렬은 <strong>Key(장르 이름) 알파벳 기준</strong>으로 정렬하는 것이라, 총 스트리밍 횟수 기준 정렬과 전혀 관계없다. 장르 순서가 완전히 잘못됐다.</li>
<li><code>playIdx</code>에 <code>plays[i] → i</code>를 저장하는 방식은 <strong>같은 스트리밍 횟수를 가진 곡이 있으면 덮어씌워진다</strong>. 인덱스를 제대로 추적할 수 없는 구조다.</li>
<li>히스토리 기준으로 <strong>20점</strong> 구간에 해당하는 풀이다.</li>
</ul>

<h3 id="3트-정답">3트 (정답)</h3>
<p>→ 두 가지 핵심 문제를 수정했다.<br />→ 첫째, 장르 정렬을 <strong>Value(총 스트리밍 횟수) 기준</strong>으로 바꿨다. <code>keySet</code>을 꺼내 <code>Comparator</code>로 커스텀 정렬.<br />→ 둘째, <code>Music</code> 객체에 <strong>인덱스(<code>idx</code>)를 직접 포함</strong>시켜 <code>playIdx</code> Map을 완전히 제거했다.</p>
<pre><code class="language-java">import java.util.*;
// 음악 클래스
class Music implements Comparable&lt;Music&gt; {
    String genre;
    int play;
    int idx;

    public Music(String genre, int play, int idx) {
        this.genre = genre;
        this.play = play;
        this.idx = idx;
    }

    @Override
    public int compareTo(Music m) {
        return m.play - this.play;
    }
}
class Solution {
    public int[] solution(String[] genres, int[] plays) {
        Map&lt;String, Integer&gt; genMap = new HashMap&lt;&gt;(); // 장르별 스트리밍 횟수 저장
        List&lt;Music&gt; playlist = new ArrayList&lt;&gt;(); // 곡별 스트리밍 횟수 저장

        for(int i=0; i&lt;plays.length; i++) {
            genMap.put(genres[i], genMap.getOrDefault(genres[i],0) + plays[i]);
            playlist.add(new Music(genres[i], plays[i], i));
        }

        // Value 기준으로 정렬
        List&lt;String&gt; keySet = new ArrayList&lt;&gt;(genMap.keySet());
        keySet.sort(new Comparator&lt;String&gt;() {
            @Override
            public int compare(String o1, String o2) {
                return genMap.get(o2) - genMap.get(o1);
            }
        });

        // 리스트에 장르별 베스트 2개 담기
        List&lt;Integer&gt; list = new ArrayList&lt;&gt;();

        for(String gen : keySet) {
            List&lt;Music&gt; tmp = new ArrayList&lt;&gt;();

            for(Music m : playlist) {
                if(gen.equals(m.genre)) tmp.add(m);
            }

            Collections.sort(tmp);

            if(tmp.size() == 1) {
                list.add(tmp.get(0).idx);
            }
            else {
                list.add(tmp.get(0).idx);
                list.add(tmp.get(1).idx);
            }
        }

        int[] answer = new int[list.size()];
        for(int i=0; i&lt;list.size(); i++) answer[i] = list.get(i);

        return answer;
    }
}</code></pre>
<h4 id="위-풀이의-결과-2">위 풀이의 결과</h4>
<ul>
<li><code>genMap.keySet()</code>을 꺼내 <code>Comparator</code>로 Value 기준 내림차순 정렬 → 장르 순서가 정확히 잡혔다.</li>
<li><code>Music</code> 객체에 <code>idx</code>를 포함시켜 같은 재생 횟수여도 인덱스를 잃지 않는다.</li>
<li>정확성 테스트 통과 🎉</li>
</ul>
